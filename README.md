# Week1-Python-Assignments
# Week 1 Python Assignments & Mini Project

## Overview

This repository contains the solutions for the Week 1 Python assignments and the mini project. All programs are written in Python and combined into a single file for easy execution.

## Assignments

### 1. Temperature Converter
- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Implemented using functions

### 2. Student Grade Calculator
- Accepts marks as input
- Calculates total and average
- Assigns grades based on the average

### 3. Even/Odd & Prime Number Checker
- Determines whether a number is Even or Odd
- Checks whether a number is Prime or Not Prime

## Mini Project

### Simple ATM Simulator

Features:
- PIN-based login
- Check account balance
- Deposit money
- Withdraw money
- Menu-driven interface
- Uses separate functions for each operation

## Python Concepts Used

- Functions
- Conditional Statements
- Loops
- User Input
- Arithmetic Operations
- Modular Programming

## How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder.
4. Run the following command:

```bash
python week1_assignments.py
```

5. Select the desired option from the menu.
